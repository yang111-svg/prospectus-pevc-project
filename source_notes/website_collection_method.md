# 网站收集方法说明

## 1. 招股说明书来源网站

### 巨潮资讯网 (cninfo.com.cn)
- **网址**: https://www.cninfo.com.cn/
- **用途**: 主板、创业板、科创板上市公司信息披露
- **检索方法**:
  1. 进入首页，点击顶部搜索框
  2. 输入"股票代码 + 公司简称 + 招股说明书"
  3. 在搜索结果中筛选"招股说明书"类别
  4. 确认文件标题包含"招股说明书"
  5. 优先选择"正式稿"或"注册稿"
- **适用板块**: 主板(SZSE/SSE)、创业板(SZSE)、科创板(SSE)

### 北京证券交易所官网 (bse.cn)
- **网址**: https://www.bse.cn/
- **用途**: 北交所上市公司信息披露
- **检索方法**:
  1. 进入首页，点击"信息披露"
  2. 在搜索框输入股票代码或公司简称
  3. 筛选"招股说明书"类别
  4. 下载PDF文件
- **适用板块**: 北交所(BSE)

## 2. 文件筛选规则

1. **确认文件类型**: 标题必须包含"招股说明书"
2. **排除非目标文件**: 上市公告书、发行公告、问询回复、发行结果公告、招股意向书（除非没有正式稿）
3. **版本优先级**: 正式稿/注册稿 > 申报稿/上会稿 > 招股意向书
4. **日期确认**: 记录招股说明书披露日期，确保是最新版本

## 3. 链接失效处理

若原始链接失效：
1. 重新到交易所或巨潮资讯检索
2. 以最新检索结果为准
3. 在 `data_sources.md` 中说明替换原因

## 4. 注意事项

- 禁止从非官方渠道下载招股说明书
- 禁止把公告、问询回复、上市公告书误当作招股说明书
- 下载后需核对文件页数和完整性
