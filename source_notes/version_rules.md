# 招股说明书文件版本规则

## 版本类型

| 版本 | 说明 | 优先级 |
|------|------|--------|
| 正式稿 | 最终发布的招股说明书 | 最高 |
| 注册稿 | 证监会注册后的版本 | 高 |
| 申报稿 | 首次提交的版本 | 中 |
| 招股意向书 | 发行前的意向性文件 | 低 |

## 选择规则

1. 优先选择**正式稿**或**注册稿**
2. 如无正式稿/注册稿，选择最新的**申报稿**
3. 避免使用**招股意向书**作为最终数据源

## 注意事项

- 北交所样本可能只有申报稿
- 需要在 `company_list.csv` 中记录 `prospectus_version` 字段
